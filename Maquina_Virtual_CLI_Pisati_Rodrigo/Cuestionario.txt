2. Describir para que sirve "apt-get".
	Este comando sirve para instalar aplicaciones que se encuentrarn en los repositorios de la distribucion del sistema operativo que tenemos.

3. Describir para que sirve "su".
	"su" es un comando que se utiliza para poder adquirir permisos de Super Usuario, para poder instalar archivos en la root o para modificar archivos instalados en la misma.
